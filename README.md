# Portfolio Website

A simple responsive portfolio website scaffolded with HTML, CSS, and JavaScript.

## Files

- `index.html` — main portfolio page
- `styles.css` — responsive styling and layout
- `script.js` — mobile menu toggle

## Get Started

1. Open `index.html` in your browser.
2. Update the text and sections with your own name, skills, and project details.
3. Optionally, add images, links, or extra sections to personalize your portfolio.
